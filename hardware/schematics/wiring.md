## 接线表

```
ESP32          →   SSD1306 OLED (I2C)
────────────────────────────────────────
GND            →   GND
3.3V           →   VCC
GPIO21 (SDA)   →   SDA
GPIO22 (SCL)   →   SCL

ESP32          →   WS2812B LED
────────────────────────────────────────
GND            →   GND
5V (VIN)       →   VCC (经过 100μF 电容滤波)
GPIO16         →   DIN (串 330Ω 电阻)

电容: 100μF 正极接 LED VCC，负极接 GND
电阻: 330Ω 串联在 GPIO16 → LED DIN 之间
```

## 注意事项

1. **WS2812B 供电**：从 ESP32 的 5V/VIN 引脚取电（USB 供电时可用），如果用 3.3V 可能导致 LED 颜色不准
2. **电容**：100μF 电解电容必须接在 LED 的 VCC-GND 之间，防止 LED 上电瞬间损坏
3. **电阻**：330Ω 限流电阻可以防止信号反射，短距离接线可省略，但建议保留
4. **I2C 上拉**：SSD1306 模组通常自带 4.7kΩ 上拉电阻，无需额外添加
5. **引脚兼容**：如使用 ESP32-C3/S3，确认 I2C 引脚映射（C3 默认 SDA=8, SCL=9，需修改固件中 `OLED_SDA`/`OLED_SCL` 宏定义）

## 扩展接线 (可选)

### USB Serial 直连模式
不需要 WiFi/MQTT，通过 USB 线直接连接电脑：
- ESP32 USB → 电脑 → Python 主机中间件检测 COM 口 → Serial 发送 JSON

### 外接按钮 (预留)
```
GPIO0 → 按钮 → GND  (模式切换 / 屏幕唤醒)
```

### LiPo 电池供电
```
LiPo 电池 → TP4056 充电模块 → ESP32 5V/VIN
                         → ESP32 GND
```
